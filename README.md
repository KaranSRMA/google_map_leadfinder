# 🚀Google Map Lead Finder Tool (only for web developers)

A simple Python-based tool to scrape business leads automatically using Selenium.

---

## 📌 Features

* Extract business leads from the web
* Automated browser scraping
* Export data (JSON / Excel)
* Easy to use CLI-based workflow

---

## ⚙️ Installation & Usage

Follow these steps to run the tool:

### Step 1: Clone the repository

```bash
git clone https://github.com/KaranSRMA/google_map_leadfinder.git
```

### Step 2: Go to project directory

```bash
cd google_map_leadfinder
```

### Step 3: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the tool

```bash
python main.py
```

### Step 5: Provide input

* Enter keyword (e.g., gym, salon, clinic)
* Enter location (e.g., city, state, country, zip)
* Choose export format (if asked)

---

## ⏳ Important Note

* The tool will automatically open a browser
* It will start collecting leads
* **Be patient** — scraping may take some time depending on data size

---

## 📁 Output

* Extracted leads will be saved in the `exports/` folder
